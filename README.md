# word_scramble_game

## Author
Saira Ali

## Description
   This project is a Python-based Word Scramble Game that allows users to guess scrambled words from different categories such as Animals, Colors and Physical Objects.

## Features
      - Uses predefined word lists for different categories
      - Allows users to add to a category
      - Allows users to choose a category
      - Randomly scrambles words
      - Checks user guesses

## Requirements

   ### Software
      - Python 3.11 or later

   ### External Libraries / Modules
      The following built-in Python modules are required:
         - random (used to shuffle and select words)

      No third-party libraries are required.

### Hardware
      - Any computer capable of running Python
      - Keyboard for user input
      - Screen/monitor for output

      Works on:
         - Windows

## Installation
      1. Install Python from https://www.python.org
      2. Download or clone the project files
      3. Open the folder in PyCharm

## How to Run
      1. Open the project in PyCharm
      2. Open the SOB5&6 word scramble Python file
      3. Click the Run button  
            OR run in terminal:
               python SOB5&6 word scramble.py

## How to Modify

   ### Add New Words
      To add new words, edit the category lists in the Python file.

      Example:

         animals = ["dog", "cat", "lion"]  
         objects = ["chair", "phone", "bottle"]

### Add New Categories
      To add a new category, create a new list:
   
      foods = ["pizza", "apple", "bread"]
   
      Then update the menu in the program.

### Change Game Settings
      You can edit inside the SOB5&6 word scramble file:
      - Number of attempts
      - Words in list
      - Number of rounds
